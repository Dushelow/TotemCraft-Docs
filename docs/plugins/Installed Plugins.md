## Плагины сервера:

- **ViaVersion** - Позволяет подключаться новым версиям клиентов к старой версии сервера.
- **ViaBackwards** - Позволяет игрокам на старых версиях заходить на текущий сервер.
- ViaRewind - Позволяет игрокам на устаревших версиях 1.7 и 1.8 также подключаться на сервер.
- **WorldEdit** - Мощный инструмент для редактирования мира (выделение, копирование, вставка и т.д.).
- **WorldGuard** - Защита регионов, флаги и правила для территорий (часто работает вместе с WorldEdit).
- **LuckPerms** - Система прав и разрешений (самый популярный плагин на права).
- **Essentials** - Основной набор команд (/spawn, /home, /warp, /tp и т.д.).
- **EssentialsSpawn** - Управление точкой спавна (работает с Essentials).
- **EssentialsXProtect** - Предотвращает урон миру от мобов и декорациям от взывчатки.
- **Geyser-Spigot** - Позволяет игрокам с Bedrock Edition заходить на Java-сервер.
- **floodgate** - Дополнение к Geyser, позволяет Bedrock-игрокам заходить без отключения онлайн-режима.
- **ProtocolLib** - Библиотека для работы с протоколом Minecraft (требуется для многих античитов и плагинов).
- **Vulcan** - Античит (использует ProtocolLib).
- **Skript** - Плагин для создания кастомных механик простыми скриптами на английском.
- **SkBee** - Аддон для Skript, добавляет много дополнительных функций.
- **PlasmoVoice** - Голосовой чат (VoIP) для игроков.
- **pv-addon-lavaplayer-lib** - Библиотека для PlasmoVoice (нужна для музыки и звуков).
- **pv-addon-discs** - Аддон для PlasmoVoice (добавляет поддержку кастомных дисков).
- **pv-addon-groups** - Аддон для PlasmoVoice (группы и разрешения в голосовом чате).
- **DiscordSRV** - Мост между сервером Minecraft и Discord (пересылка сообщений чата).
- **TelegramBridge** - Мост между сервером и Telegram (пересылка сообщений).
- **CoreProtect** - Логирование и защита блоков (можно откатывать гриф).
- **LWC** - Защита сундуков, дверей и других блоков (локальная защита).
- **AuthMe** - Система регистрации и авторизации игроков (логин/пароль).
- **SkinsRestorer** - Восстановление и смена скинов игроков на сервере.
- **GSit** - Плагин для сидения и лежания на ступеньках, коврах и т.д.
- **InvisibleItemFrames** - Делает рамки с предметами невидимыми.
- **DeathChest** - Создаёт сундук смерти с вещами игрока после смерти.
- **InventoryRollbackPlus** - Откат инвентаря игроков (восстановление вещей после смерти/бага).
- **InvSeePlusPlus** - Просмотр и редактирование инвентаря других игроков.
- **SimpleVanish** - Режим невидимости для администраторов.
- **AdvancedBanX** - Система банов, мутов и киков.
- **Pl-Hide-Free** - Скрытие плагинов от игроков (анти-утечка списка плагинов).
- **SimpleClans** - Система кланов/гильдий.
- **PlayTime** - Отслеживание игрового времени и выдача наград за него.
- **UltimateAutoRestart** - Автоматический рестарт сервера по расписанию.
- **Chunky** - Быстрая и безопасная прегенерация чанков мира.
- **ChunkSpawnerLimiter** - Ограничение количества мобов и блоков в чанках (оптимизация).
- **VillagerLobotimizer** - Отключает ИИ в торговых залах (сильно улучшает производительность).
- **RedstoneClockDetector** - Обнаружение лаг-машин на редстоуне.
- **WorldEditSelectionVisualizer** - Визуализация выделения WorldEdit частицами.
- **WGLecternFlags** - Добавляет флаги WorldGuard для управления лектернами.
- **NBTAPI** - Библиотека для работы с NBT-данными (требуется многими плагинами).
- **packetevents** - Библиотека для работы с пакетами (используется Vulcan и другими).
- **packetevents** - Библиотека для работы с пакетами (используется Vulcan и другими).
- **EmoteCraft** - Поддержка мода на анимации игроков.
