### 📜 Сертификаты

**Один сертификат** — **Universal SSL** (бесплатный, автоматический):

| Параметр                | Значение |
|------------------------|----------|
| **Тип**                | Universal (автоматический) |
| **Покрытие**           | `[YOUR_DOMAIN]` + `*. [YOUR_DOMAIN]` (wildcard) |
| **УЦ**                 | Google Trust Services |
| **Алгоритм**           | ECDSA + SHA-256 |
| **Валидация**          | DV (через TXT) |
| **Срок действия**      | 90 дней |
| **Статус**             | ✅ Active |

---

### Зачем и почему эти сертификаты?

1. **Universal SSL** — автоматически создаётся и продлевается Cloudflare при добавлении домена. Используется для шифрования между пользователями и edge-серверами Cloudflare.
2. **Wildcard** — покрывает все поддомены одним сертификатом.
3. **Режим Full** — шифрует весь путь, но **не проверяет** сертификат на вашем origin-сервере.  
   Для полной проверки используйте **Full (Strict)** + Origin Certificate от Cloudflare.

---

### ⚠️ Рекомендации (обязательно к выполнению)

1. **Включить Always Use HTTPS** — автоматический редирект на HTTPS.
2. **Поднять минимальную TLS до 1.2** (лучше 1.3).
3. **Включить HSTS** (после теста!).
4. **Включить DNSSEC** — защита от DNS-спуфинга.

### 🔑 Расшифровка шаблонов (замените перед использованием)

| Шаблон                    | Что заменить на                                                                 |
|---------------------------|---------------------------------------------------------------------------------|
| `[YOUR_DOMAIN]`           | Основной домен (например: `totemcraft.net`)                                    |
| `[IP_PLAY_SERVER]`        | IP-адрес сервера для `play.` поддомена, то есть основной VPS сервер с игрой    |
| `[IP_RU_SERVER]`          | IP-адрес русского / альтернативного сервера для reverse proxy                  |
| `[IP_SITE_SERVER]`        | IP-адрес веб-сервера (на котором висят сайт, wiki и т.д.)            |
| `[RECORD_ID_1]` … `[RECORD_ID_8]` | Внутренние ID записей Cloudflare (можно оставить как есть или удалить столбец) |
| `[DATE_CREATION]`         | Дата создания зоны в Cloudflare                                                 |
| `[DATE_ACTIVATION]`       | Дата активации зоны                                                             |
| `[NS1]` и `[NS2]`         | Имена nameserver’ов Cloudflare, назначенных на домен (например: `garrett`, `nelci`) |
