# TotemCraft

**Высокопроизводительный ванильный Minecraft-сервер**

На сервере нет привилегий и спама в чате, а также нет приватов, доната, вайпов, телепортов и прочих лишних элементов. Проект создан с любовью к качественному и чистому Minecraft - максимально приближённый к ванилле геймплей.

Этот репозиторий - открытая база знаний и полный набор конфигураций для создания аналогичного высокопроизводительного ванильного сервера. Здесь собрано всё, что было получено от знакомых администраторов крупных проектов, программистов, технической документации и открытых источников, а также из личного опыта администрирования и разработки игровых серверов - более четырёх лет.

Пользуйтесь на здоровье!

---

## Особенности проекта

- Ядро Purpur 1.21.11
- Кроссплатформенный: поддерживает Java и Bedrock, включая мобильные устройства
- Поддержка старых версий Java (до 1.21.11) и актуальных клиентов (26w02+)
- Глубокая оптимизация производительности на всех уровнях, включая ядро Linux - сервер держит 19+ TPS при 20 онлайне
- Современный античит, Anti-Xray, полностью рандомизированные сиды структур
- Автоматическая система регистрации через Telegram-бота
- Вход для RU-игроков через Reverse Proxy в обход блокировок
- Интернациональная аудитория: Европа, англоязычные страны и всё постсоветское пространство
- Надёжная система бэкапов
- Полноценная защита от грифа
- Подробная техническая документация

## Структура репозитория

| Папка                        | Содержимое                                    |
|------------------------------|-----------------------------------------------|
| `docs/`                      | Полная техническая документация               |
| `minecraft/core-configs/`    | Конфиги ядра Minecraft (Paper, Purpur и т.д.) |
| `minecraft/plugins-configs/` | Конфиги установленных плагинов                |
| `minecraft/custom-plugins/`  | Кастомные плагины сервера                     |
| `system/scripts/`            | Bash-скрипты управления сервером              |
| `system/network/`            | Конфигурации nginx и reverse proxy            |
| `bots/`                      | Код Telegram и Discord ботов                  |
