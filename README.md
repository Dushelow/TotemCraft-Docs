# TotemCraft

**Высокопроизводительный ванильный Minecraft-сервер**

На сервере нет привелегий и спама в чате, также нет приватов, доната, вайпов, телепортов и прочих лишних элементов. Проект создан с любовью к качественному и чистому Minecraft'у - максимально приближенный к ванилле геймплей. 

Этот гит - открытая база знаний и полный набор конфигураций для создания аналогичного высокопроизводительного ванильного сервера. Здесь я собрал то, что узнал сам. Информацию черпал сначала от знакомых администраторов крупных проектов, программистов, технической документации и открытых источников в интернете, а затем из личного опыта администрирования и разработки игровых серверов - более четырёх лет.

Пользуйтесь на здоровье! 

---

## Особенности проекта

- Ядро purpur 1.21.11
- Сервер кроссплатформенный, поддерживает Java и Bedrock, включая мобильные устройства 
- Есть поддержика старых версии Java (до 1.21.11) и актуальных клиентов (26.2+)
- Глубокая оптимизация производительности на всех этапах, в том числе на уровне ядра Linux - сервер держит 19+ TPS при 20 онлайна
- Современный античит, Anti-Xray, полностью рандомизированные сиды структур
- Автоматическая система регистрации через Telegram-бота
- Вход для RU игроков через Reverse Proxy в обход блокировок
- Аудитория проекта интернациональная и объединяет игроков из Европы, англоязычных стран и всего постсоветского пространства
- Надёжная система бэкапов
- Полноценная защита от грифа
- Подробная техническая документация

## Структура репозитория

| Папка              | Содержимое                                      |
|--------------------|-------------------------------------------------|
| `docs/`            | Полная техническая документация                |
| `minecraft/core-configs/`  | Конфиги ядра Minecraft (paper, purpur и т.д.) |
| `minecraft/plugins-configs/`  | Конфиги установленных плагинов                 |
| `minecraft/custom-plugins/`  | Кастомные плагины сервера                 |
| `system/scripts/`         | Bash-скрипты управления сервером               |
| `system/network/`           | Конфигурации nginx и reverse proxy             |
| `bots/`             | Код Telegram и Discord ботов                       |
